Данный прект содержит содержит UVM тестбенч для контроллера управления вендингового автомата.
В папке "информация про проект" содержится спецификация устройства, тест-план и текущий прогресс.
Тестируемый модуль находится в папке dut, тестбенч находится в папке tb.
Проект реализован на UVM 1.2
Для работы необходимо скомпилировать файлы (именно в таком порядке):  vending_machine.sv vm_pkg.sv top.sv
Для использования кастомного report_server необходимо при компиляции добавить флаг +define+USE_CUSTOM_REPORT_SERVER. Чтобы он выводил в заголовки файлов логов время начала симуляции, то необходимо дополнительно скомпилировать файл c_functions.c (до компиляции файлов sv) и добавить флаг +define+USE_C_FUNCTIONS
